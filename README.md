\# Cliente SOAP Multilenguaje



Este repositorio contiene ejemplos de consumo de un servicio web SOAP público utilizando diferentes lenguajes de programación del lado del servidor.



El servicio utilizado es NumberConversion:

https://www.dataaccess.com/webservicesserver/NumberConversion.wso?WSDL



Cada lenguaje contiene tres versiones:



1\. Cliente SOAP que convierte un número a letras en inglés.

2\. Cliente SOAP con traducción del resultado de inglés a español.

3\. Aplicación web que convierte un número a letras en español.
